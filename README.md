# React-zero-to-one

从零到一搭建 React + TS 项目

## Project setup

```bash
$ npm install
```

### Compiles and hot-reloads for development

```bash
$ npm run serve
```

### Compiles for development

```bash
$ npm run build:dev
```

### Compiles and minifies for production

```bash
$ npm run build:prod
```
